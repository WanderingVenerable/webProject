About Me

Hey there! I'm a high school student who's passionate about coding, and this is one of my first projects I've built using HTML, CSS, and JavaScript. It's a simple number guessing game that I put together to challenge myself and have some fun with coding.

The Idea:


The game randomly generates a number, and your job is to guess it. With each guess, it'll tell you if you're getting close or if you're way off. It's a straightforward concept, but it's been a cool way to practice my skills and see how code can create something interactive and enjoyable.
